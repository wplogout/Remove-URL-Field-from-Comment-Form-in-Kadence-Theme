This plugin helps to remove URL Field from comment form in Kadence Theme.

== Description ==
The plugin allows you to disable/Remove the URL/Website field from Comment form in Kadence Theme. The Remove URL Field from Comment Form in Kadence Theme plugin works in both free version and Premium Version of Kadence Theme.

Features:

1. Remove URL Field in Free Version of Kadence Comment Box.
2. Remove Website Field from Premium Version of Kadence Comment Box.

== Installation ==
#### From within WordPress
1. Visit 'Plugins > Add New'
1. Search 'Remove URL Field from Comment Form in Kadence Theme'
1. Install and Activate the plugin.

#### Manually
1. Download and unzip the file.
1. Upload the folder Remove URL Field from Comment Form in Kadence Theme plugin into 'wp-content/plugins/' folder.
1. Activate from the plugin page and Enjoy.
